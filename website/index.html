Get Started in 60 Seconds

1

Install

Install via Go or build from source.

go install github.com/CloudPulse-HQ/tfwatch@latest
2

Start the observability stack

Spins up OTEL Collector, Prometheus, and Grafana with dashboards pre-loaded.

docker compose -f deploy/docker-compose.yml up -d
3

Run tfwatch

Point it at any Terraform directory. Backend is auto-detected.

tfwatch --dir ./your-terraform-repo
4

View in Grafana

Open localhost:3000 ‚Äî both dashboards are automatically available.
